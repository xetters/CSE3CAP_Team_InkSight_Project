How It Works

Privacy Features

  • 100% Local Processing - All analysis happens on our own servers
  • Zero Data Storage - Files are processed in memory only (not stored) and immediately discarded
  • No Tracking - No analytics, no logging, no external connections

How to Use

  1. Upload - Select a text file (.txt, .docx, or .md max 5MB)
  2. Analyze - Choose your analysis tool: Word Frequency, Keyness Statistics, or Semantic Clustering
  3. Review - Use our advanced dashboard to view your results
  4. Download - Export/download results as standalone HTML reports (optional)
  5. Clear - Select clear or simply navigate away to refresh/remove all results when done
