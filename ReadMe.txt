# Running This App on Another PC

******************************************

## Prerequisites

Ensure the following software dependencies and versions are installed:

- Node 18+
- Npm v9+
- Python 3.9+

******************************************

## Installation

Run `npm install` in the root folder and in the two main subfolders (`api` and `frontend`)

******************************************
